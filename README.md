# exsteal
Extract Stealer Logs

The script will help you to extract a specific domain or multiple domain from stealer logs

Usage : \
``` git clone https://github.com/mhrsq/exsteal ```\
``` cd exsteal ```\
``` chmod +x extract.sh ```\
``` bash extract.sh ```\
or\
``` ./extract.sh ```
